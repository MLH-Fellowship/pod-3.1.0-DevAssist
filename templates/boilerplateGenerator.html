Boilerplate Generator

Bored of writing code for full stack appliaction from scratch. Use these boilerplates for your next web application !!
Simply download the zip file and get a copy of boilerplate.
Then run docker-compose build .
This spins up Compose and builds a local development environment according to our specifications in docker-compose.yml.
After the containers have been built , run docker-compose up.

React-Flask Boilerplate

Development environment for a web application running a React front-end and Flask API back-end.

React-Django Boilerplate

Development environment for a web application running a React front-end and Django back-end.

React-Node Boilerplate

Development environment for a web application running a React front-end, Node(express) back-end and Mongo database.
